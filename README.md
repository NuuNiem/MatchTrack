# MatchTrack

Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.

Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan otteluita, joita on käynyt katsomassa.

Käyttäjä pystyy lisäämään ottelulle tietoja, kuten päivämäärän, vastustajan, tuloksen, sijainnin ja mahdollisia muistiinpanoja ottelusta.

Käyttäjä näkee sovellukseen lisäämänsä ottelut listana.

Käyttäjä pystyy etsimään ja suodattamaan otteluita vastustajan, päivämäärän tai sijainnin perusteella.

Sovelluksessa on käyttäjäsivu, joka näyttää tilastoja, kuten katsottujen otteluiden lukumäärän, suosituimmat joukkueet ja käyntimäärän eri stadioneilla.

Käyttäjä pystyy tarkastelemaan tilastoja myös graafisessa muodossa.

Sovellus tallentaa tiedot tietokantaan ja näyttää ne käyttäjäkohtaisesti.
